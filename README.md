# file-viewer-cs

汎用的なファイルビューアの実装です。  

独自のファイル圧縮形式などの中身を確認するビューアとして、  
一時フォルダに展開したファイルの中身を表示するという使用方法を想定しています。

- [x] ファイルをコピーする
- [x] ファイルをドラッグ＆ドロップでコピーする
- [x] ファイルをダブルクリックすると、関連付けられたアプリケーションを開く
- [x] リストビューの表示切替
- [ ] 画像のサムネイル表示

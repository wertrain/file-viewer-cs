# file-viewer-cs

汎用的なファイルビューアの実装です。  

独自のファイル圧縮形式などを使用する場合のビューアとして、  
一時フォルダに展開したファイルの中身を確認する使用方法を想定しています。

- [x] ファイルをコピーする
- [ ] ファイルをドラッグ＆ドロップでコピーする
- [x] ファイルをダブルクリックすると、関連付けられたアプリケーションを開く
- [x] リストビューの表示切替
